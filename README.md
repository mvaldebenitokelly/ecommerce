# Portafolio Módulo 2 - Front End

## 1. Descripción
Proyecto MVP de e-commerce frontend desarrollado con **HTML5 semántico**, **Bootstrap 5** y **JavaScript básico**.

La temática elegida es una tienda ficticia llamada **CosmoStore**, enfocada en productos de tecnología espacial.

## 2. Objetivo académico
Este proyecto fue construido para demostrar:

- uso correcto de etiquetas semánticas (`header`, `nav`, `main`, `section`, `article`, `footer`)
- diseño responsivo con Bootstrap y enfoque mobile-first
- manipulación del DOM con JavaScript
- contador de carrito actualizado en tiempo real
- navegación entre páginas del MVP
- versionado del proyecto con Git y publicación en GitHub

## 3. Estructura del proyecto

```text
 ecommerce-frontend-m2/
 ├── index.html
 ├── detalle.html
 ├── carrito.html
 ├── README.md
 └── assets/
     ├── css/
     │   └── styles.css
     └── js/
         └── app.js
```

## 4. Tecnologías utilizadas
- HTML5
- Bootstrap 5 por CDN
- CSS3
- JavaScript
- localStorage
- Git / GitHub

## 5. Cómo ejecutar el proyecto
1. Descargar o clonar el repositorio.
2. Abrir la carpeta del proyecto en Visual Studio Code.
3. Ejecutar con **Live Server** o abrir `index.html` en el navegador.

## 6. Funcionalidades implementadas
- Home con grilla de productos
- Página de detalle del producto
- Carrito simulado
- Contador en navbar
- Persistencia simple con `localStorage`
- Diseño adaptable para móvil y escritorio
